# Calculadora de Potència i Cost Energètic

Projecte educatiu per a la Unitat 1 (Energia, Potència i Rendiment).

## Contingut
- `index.html` : interfície principal
- `style.css` : estils
- `script.js` : lògica de càlcul

## Com provar-ho localment
1. Clona el repositori.
2. Obre `index.html` en un navegador o utilitza un servidor local.

## Com desplegar a Replit
- Importa el repo des de GitHub a Replit (New Repl → Import from GitHub).
